# Devaria - aula 25 - JS - Aula 2 de linguagem de programação

### Estrutura de Repetição
- Declaração do For
```
for ([expressaoInicial]; [condicao]; [incremento])
  declaracao
```
- Declaração do while
```
while (condicao)
  declaracao

```
- Declaração do...while
```
do
  declaracao
while (condicao);

```
